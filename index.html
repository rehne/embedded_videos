Allgemeines

Das Thema dieser Seite beschäftigt sich mit der Einbettung/Einbindung von Videos auf Webseiten. Wir möchten euch versuchen zu erklären wie man Videos mithilfe von verschiedenen Techniken in Websites einbindet.

Einfache Einbindung (YouTube)

Die simpelste Methode der Einbettung ist das Einbinden von Videos bekannten Videoportalen wie Youtube oder Vimeo. Diese stellen uns einen Iframe-Code zur Verfügung, den wir in unser Markup der HTML Datei der Seite einbringen.


Beispiel



Der Iframe befindet sich unter dem jeweiligen Video der Seite. Mit einem Klick auf Teilen - Einbetten erhält man den Code.


Code:

< iframe width="640" height="360" src="https://www.youtube.com/embed/YgM8HVoG2G8" frameborder="0" allowfullscreen>
< /iframe>

Erläuterung:

Das Element Iframe bzw. Inlineframe ist ein HTML-Element, welches als Objekt innerhalb einer Webseite plaziert wird. In unserem Beispiel das Video welches eingebunden wird.
Das Attribut width gibt Breite und height die Höhe des Frames an.
Source bzw. src steht für die Quelle der Datei und frameborder=0 gibt an, dass unser Rahmen keinen Rand besitzt.

Weitere Iframe Attribute sind auf W3Schools.com zu finden

HTML5 video Element

Die Hyper Markup Language 5 beinhaltet das Multimedia-Element < video>, welches benutzt werden kann um unser Video ohne zusätzliche Plug-Ins einzubetten. Vorraussetzung für das Benutzen von HTML5 ist das der Browser auf der möglichst aktuellen Version ist.

  • Firefox 3.0 (oder höher)
  • Safari 3.0 (oder höher)
  • Opera 10.0 (oder höher)
  • Chrome 3.0 (oder höher)

Beispiel

< video "width="100%" height="480" controls preload="auto" poster=" thumbnail.jpg">
< source src=" meinvideo.webm" type="video/webm">< /source>
< source src=" meinvideo.mp4" type="video/mp4">< /source>
< /video>

Erläuterung:

Das Element < video> ist unser Objekt welchem wir verschiedene Parameter zuweisen können.
controls fügt eine Reihe von Videosteuerelementen hinzu.
preload definiert erforderliche Puffermenge des Videos.
poster gibt das Platzhalterbild an, das angezeigt wird bis der Inhalt geladen und gestartet wird.

Weitere video Attribute sind auf W3Schools.com zu finden

Einbindung durch Zusatztechnologien ( Plug-Ins )

Flash

Erlaubt es uns der Browser nicht das Video nativ mithilfe des < video>-tags einzubinden schafft der Adobe Flash Player abhilfe. Dieses Browser Plug-In ermöglicht uns durch verschiedene Methoden Videos einzubetten.

Quicktime

Eine Alternative zu dem Adobe Flash Player bietet uns Apple's Quicktime. Genau wie bei der Flasheinbindung kann das Video über einen < object> oder < embed> tag angezeigt werden.
